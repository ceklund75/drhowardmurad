# Dr. Howard Murad — Website Migration

End‑to‑end migration of the Dr. Howard Murad marketing site from a legacy WordPress/Uncode theme to a modern headless stack. I designed and implemented the entire system: frontend, data layer, and deployment.

**Tech:** Next.js · TypeScript · Tailwind CSS · GraphQL · WordPress (headless)

## Highlights

- Rebuilt public site and blog as a headless Next.js app backed by WordPress.
- Implemented responsive, accessible layout with reusable UI components.
- Added GraphQL data layer for posts, pages, SEO metadata, and media.
- Improved authoring workflow with preview tooling and structured content.
- Deployed to a Next.js‑compatible host with environment‑based configuration.

## Repository Structure

- `/app` – Next.js application (frontend, GraphQL client, UI components).
- `/docs` – Project notes, architecture decisions, and content/deployment docs.
