## Color Reference Guide

**Note:** All gray backgrounds reference one of two options:
- **Light Gray (Light 2):** Uncode ID `color-gyho`, hex `#eaeaea`
- **Light 1 (Light Gray):** Uncode ID `color-light1`, hex `#f5f5f5`

## Semantic Color Tokens (Tailwind)

Based on the page color mapping, define these semantic tokens in `tailwind.config.ts`:

```ts
theme: {
  extend: {
    colors: {
      // Full palette in colors.md

      // Semantic brand colors
      brand: {
        // Global defaults
        text: '#464646',                // Gray
        background: '#ffffff',          // White
        border: '#dddddd',              // Light 3
        footerAccent: '#fdc405',        // Gold

        // Page-level theme colors
        innovator: '#015dcf',           // Blue Heading
        holistic: '#c3158a',            // Pink
        lifeStory: '#91298c',           // Medium Purple
        books: '#6baf1a',               // Green
        publications: '#09b0a2',        // Cyan

        // Home page section colors
        homeSection1: '#c3158a',        // Pink
        homeSection2: '#015dcf',        // Blue Heading
        homeSection3: '#fc5c0e',        // Orange
        homeSection4: '#91298c',        // Medium Purple

        // Background tints
        light1Bg: '#f5f5f5',            // Light 1
        light2Bg: '#eaeaea',            // Light Gray (Light 2)
        lightBlueBg: '#d9e7f8',         // Light Blue Background

        // Hero accent colors
        holisticHeroAccent: '#fc5c0e',  // Orange (for Holistic hero)
      },
    },
  },
}
```
